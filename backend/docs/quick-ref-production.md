# NexaForge ERP — Quick Reference: Production Team

**Login:** your work email  **Role:** `user` (view + log) or `senior` (approve + add steps)

---

## Daily tasks

| Task | Where | Steps |
|---|---|---|
| Check your routing steps | Production → select project → Routing | Steps listed with status: Pending / In Progress / Complete |
| Start a routing step | Routing → click step → Update Status → In Progress | Enter start time |
| Complete a routing step | Routing → click step → Update Status → Complete | Enter actual hours + notes |
| Raise a Material Request | Production → Material Requests → New MR | Fill: item, qty, unit, project, needed-by date |
| View BOM for your project | Projects → open project → BOM tab | Tree view — click any node for detail |
| Check MRP shortages | Production → MRP → Run for Project | Shows: available stock vs. required qty |

---

## Raising a Material Request

1. Click **Material Requests** in the sidebar
2. Click **New Request**
3. Select the project
4. Enter item description, quantity, unit of measure
5. Set the date you need it by
6. Click **Submit** — Store team will receive it

---

## Things you CANNOT do

- Delete projects or BOM items (manager+ only)
- Sign off ITP inspection steps (QC team only)
- Create invoices (Finance only)

---

## Common issues

| Problem | Fix |
|---|---|
| Routing step won't move to Complete | Check all previous steps are marked In Progress or Complete first |
| MRP shows shortage but stock exists | Ask Store to confirm GRN has been logged and quarantine cleared |
| Can't see a project | Ask your manager to verify you are assigned to the project |
