# Projects Module

**Purpose**: Central management of project entities and cross-module tracking.

## Key Features
- **Active Project Selector**: Global context switching for project-specific data.
- **Phase Tracking**: Monitoring projects from Initiation to Dispatch.
- **Entity Creation**: Automated project setup from won CRM opportunities.
